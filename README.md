# 🚀 React Portfolio

![React](https://img.shields.io/badge/React-18-blue?logo=react) ![Vite](https://img.shields.io/badge/Vite-5-purple?logo=vite)  ![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow?logo=javascript)  ![Status](https://img.shields.io/badge/status-finalizado-brightgreen)

Meu portfólio pessoal desenvolvido com **React + Vite**, com foco em performance, organização de componentes e design moderno.

---

## 📌 Sobre o Projeto

Este projeto foi criado com o objetivo de:

- Apresentar meus projetos
- Mostrar minhas habilidades técnicas
- Servir como portfólio profissional
- Praticar React e boas práticas de organização
- Aplicar conceitos de componentização e reutilização de código

---

## 🛠️ Tecnologias Utilizadas

- ⚛️ React  
- ⚡ Vite  
- 🎨 CSS Modules  
- 📦 Node.js  
- 💻 JavaScript (ES6+)  

---

## ✨ Funcionalidades

- 📂 Seção de projetos com descrição e tecnologias utilizadas  
- 👤 Seção sobre mim  
- 📱 Layout responsivo  
- 🔗 Links para redes sociais  
- ⚡ Aplicação otimizada para carregamento rápido
